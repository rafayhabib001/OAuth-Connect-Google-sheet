# 📊 Create Google Sheet using n8n

## 🔹 Overview

This n8n workflow creates a new Google Spreadsheet with a single click.

## ⚙️ How It Works

1. Manual trigger is executed
2. A new Google Sheet is automatically created

## 🛠 Setup Instructions

1. Import the workflow into n8n
2. Connect your Google Sheets account
3. Click "Execute Workflow"

## 💡 Use Case

* Quickly create spreadsheets
* Automation setups
* Beginner n8n practice

## ⚠️ Note

This is a safe template. No credentials are included.
